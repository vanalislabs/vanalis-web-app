# PLAN_000003: Localization - Translate All Indonesian Text to English

## Deskripsi Singkat

Mengubah semua teks bahasa Indonesia yang masih ada di aplikasi menjadi bahasa Inggris sesuai dengan Product Brief yang menyatakan bahwa aplikasi harus menggunakan bahasa Inggris untuk semua teks UI, label, pesan, dan konten. 

**Konteks dari User:**
> "Kalo kamu cek di produk brief itu bahasa localization yang digunakan itu bahasa inggris kan ya, nah ini aku cek aplikasi nya dari browser masih ada label2 yang menggunakan bahasa indonesia. Bisakah kamu bantu untuk plan perubahan bahasa nya?"

## Algoritma / Pendekatan

### Fase 1: Identifikasi (Discovery)
1. Scan semua file `.tsx` di folder `src/pages/` dan `src/components/`
2. Cari pola teks bahasa Indonesia menggunakan regex pattern:
   - Kata-kata umum: "untuk", "dengan", "yang", "akan", "sampai", "oleh", "sebagian", "bukan", "atau", "jika", "gambar", "data", "review", "curator", "accessible", "locked", "approve", "reject", "distributed", "reward", "submission", "ditolak", "upload", "preview", "full", "dataset", "storage", "dibuka", "tetap", "representatif", "lengkap", "Catatan", "contoh", "berisi", "bisa", "verifikasi", "kualitas"
   - Frasa umum: "di-lock", "di-approve", "untuk review", "sampai approve", "oleh curator"
3. Buat daftar lengkap semua teks bahasa Indonesia yang ditemukan

### Fase 2: Terjemahan (Translation)
1. Untuk setiap teks bahasa Indonesia yang ditemukan:
   - Identifikasi konteks penggunaan (label, helper text, description, error message, dll)
   - Terjemahkan ke bahasa Inggris yang natural dan sesuai konteks
   - Pastikan terjemahan konsisten dengan terminologi yang sudah ada di aplikasi
   - Pertahankan format HTML jika ada (seperti `<strong>`, dll)

### Fase 3: Update File (Implementation)
1. Update setiap file yang mengandung teks bahasa Indonesia
2. Ganti semua string literal bahasa Indonesia dengan terjemahan bahasa Inggris
3. Pastikan tidak ada teks yang terlewat

### Fase 4: Verifikasi (Verification)
1. Test aplikasi di browser untuk memastikan semua teks sudah dalam bahasa Inggris
2. Periksa semua halaman dan komponen untuk memastikan tidak ada teks bahasa Indonesia yang tersisa
3. Pastikan format dan styling tetap konsisten setelah perubahan
